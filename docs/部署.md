# 腾讯云 EdgeOne Makers 使用指南

## 参赛提交（必做）

> **完成 EdgeOne 部署不等于成功参赛。** 部署并拿到项目预览链接后，必须前往[大赛官网](https://portal.learn.woa.com/pages/activityLaodingPage/index.html?scheme_type=aiBlock&from=xx#topics)提交作品并填写链接，才算完成参赛。

建议个人作者尽早上传可运行的 Demo 或说明文档；作品提交后仍可继续优化和修改，也便于参与早鸟激励等活动。

- **提交入口**：[大赛官网](https://portal.learn.woa.com/pages/activityLaodingPage/index.html?scheme_type=aiBlock&from=xx#topics)
- **官方部署指引**：[EdgeOne 部署指南](https://doc.weixin.qq.com/doc/w3_AR0AxQbWANECNicr5O9cwSF2XjOh0?scode=AJEAIQdfAAoyhAsUkmAR0AxQbWANE)

腾讯云 EdgeOne Makers 提供参赛项目的部署上线服务，支持Agent、前端页面、动态 API 。若您已经将参赛项目部署在 EdgeOne Makers，可在提交作品时附上项目预览链接。

一、通过 Skill 快速部署（以 Workbuddy 为例）
1.在 Workbuddy 内完成Agent的本地开发与调试后，可以安装我们提供的 Skill 快速部署：
npx skills add TencentEdgeOne/edgeone-makers-tools

Skill 会先检查是否已安装 EdgeOne CLI，若没有则自动配置。安装完毕后，请选择使用腾讯云个人的中国站账号或国际站账号进行登录（推荐中国站）。

通常情况下，Workbuddy 会在浏览器中弹出腾讯云登录页，完成授权即可。如果处于特殊环境或者浏览器无法弹出，则会切换到 Token 登录模式，可以自行在 Makers 控制台的设置中生成 API Token，粘贴回来即完成授权。

登录完成后，Skill 将自动完成首次部署。部署完毕后，请手动打开 EdgeOne 控制台，在项目设置页里上传环境变量，并再重新部署一次，使环境变量生效。

部署成功后，可点击项目右上角「预览」按钮，获取预览链接。

更多关于 EdgeOne Makers Skill 的说明，请参考：https://cloud.tencent.com/document/product/1552/129329

二、前往 EdgeOne Makers 控制台进行部署
1.Agent 快速部署指引：https://cloud.tencent.com/document/product/1552/132786
2.Web 应用快速部署指引：
●导入 Git 仓库：https://cloud.tencent.com/document/product/1552/127369
●直接上传：https://cloud.tencent.com/document/product/1552/127371

三、特别说明
1.为确保内容合规，默认的项目预览链接具有访问时效限制，具体规则如下：
●包含中国大陆加速区域：有效期 3 小时，过期后需在项目概览右上角点击“预览”按钮更新
●不含中国大陆加速区域：无时效限制（仅限非中国大陆网络访问）

2.为确保比赛期间，部署在 EdgeOne Makers 的参赛项目可长期访问，我们将收集您的项目预览链接，生成可长期访问的专用链接。如有需要，请填写https://doc.weixin.qq.com/forms/AJEAIQdfAAoAVMAxAZlAFoCNNs69fqmPf

3.如需对外分享、实现项目长期访问，建议绑定自定义域名，详情参考https://cloud.tencent.com/document/product/1552/127404 。
